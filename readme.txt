稚内→肥前山口の最長経路算出用ファイルです。

北海道内(稚内→木古内)、本州内(新青森→小倉)、九州内(小倉→肥前山口)に分けて算出可能

pythonのコードファイルにおいて
hokkaido.csvを読み込み、("稚内","木古内")で始点と終点を定めれば北海道内
honshu.csvを読み込み、("新青森","小倉")で始点と終点を定めれば本州内
kyushu.csvを読み込み、("小倉","肥前山口２")で始点と終点を定めれば九州内
の最長経路をそれぞれ算出可能です。
